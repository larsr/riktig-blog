riktig #11 — The first ε

Eleventh post documenting the riktig project. Analysis begins: sequence limits over normed spaces, with the proofs shown blackboard-style and as they live in the repository.

What we are doing, and why. Verified software in Coq, built for modifiability. Post #8 bought the order and the norm; today we spend them. This is the first genuinely analytic file in the tree — the first ε, the first N, the first ε/2 argument — and it needs no real numbers and no completeness: limits are defined over any normed space over any ordered field. Completeness is only for proving limits exist; that’s a later post.

The definition

A sequence x0,x1,x2,…x_0, x_1, x_2, \dots of vectors converges to LL when it eventually stays inside every ε-ball around LL (Theory/Limits.v):

Definition seq_lim (x : nat -> V) (L : V) : Prop :=
  forall eps : F, 0 < eps ->
    exists N : nat, forall n : nat, (N <= n)%nat -> ‖ x n + - L ‖ < eps.

Read it aloud and it is the sentence from your first analysis course. The distance is measured by the norm from post #8; the ε lives in the ordered scalar field.

Theorem: limits are unique — the ε/2 argument

Blackboard. Suppose xn→L1x_n \to L_1 and xn→L2x_n \to L_2. For any ε>0\varepsilon > 0, go far enough that both ∥xn−L1∥<ε/2\|x_n - L_1\| < \varepsilon/2 and ∥xn−L2∥<ε/2\|x_n - L_2\| < \varepsilon/2; then by the triangle inequality

∥L1−L2∥≤∥L1−xn∥+∥xn−L2∥<ε2+ε2=ε.\|L_1 - L_2\| \le \|L_1 - x_n\| + \|x_n - L_2\| < \tfrac{\varepsilon}{2} + \tfrac{\varepsilon}{2} = \varepsilon.

So the nonnegative quantity ∥L1−L2∥\|L_1 - L_2\| is below every positive ε, hence zero; and the norm is definite, so L1=L2L_1 = L_2. ∎

Repository. Every phrase above is a named lemma, and the proof is their chain. The lemma types to check it against:

blackboard phrase lemma
“go far enough that both…” Nat.max of the two N’s
“ε/2 + ε/2 = ε” half_sum : half a + half a == a
“ε/2 is still positive” half_pos : 0 < a -> 0 < half a
triangle via the midpoint norm_sub_triangle : ‖u + -w‖ <= ‖u + -v‖ + ‖v + -w‖
“below every positive ε ⇒ zero” nonneg_below_all_pos_is_zero
“norm is definite” norm_def : ‖u‖ == 0 -> u == 0

That toolbox didn’t exist last week. Building this file forced a whole layer of order arithmetic into Theory/Order.v — monotonicity of + and * in all strict/non-strict mixtures, 0 < 1, 0 < 2, positivity of inverses, halves. Each is bachelor-level; each is now a permanent, reusable rewrite. This is the project thesis acting on mathematics itself: the theorem obliges you to establish a few local facts, and the facts outlive the theorem.

The fine print your lecturer skipped

One step above is subtler than it looks. “Below every positive ε, hence zero” — how do you conclude that, constructively? If the order were decidable you’d case-split on x == 0. It isn’t (think computable reals). What the halving argument actually yields is “x cannot fail to be ≤ 0” — a double negation. So OrderedField gained one honest law:

le_stable : forall a b : F, ~ ~ a <= b -> a <= b

The order can’t be undecided forever. Every intended model pays nothing for it: decidable orders like ℚ trivially, classical ℝ trivially, and constructive Cauchy reals define ≤ negatively (a ≤ b := ¬ b < a), so stability is built in. This is the kind of line an informal proof never writes down — and exactly the kind a machine-checked one must.

The algebra of limits

With uniqueness done, the workhorse lemmas follow the same pattern — a rearrangement lemma plus the triangle:

xn→L,yn→M⇒xn+yn→L+Mxn→L⇒axn→aLx_n \to L,\; y_n \to M \;\Rightarrow\; x_n + y_n \to L + M \qquad\qquad x_n \to L \;\Rightarrow\; a\,x_n \to a\,L

lim_add rides norm_sub_pair (∥(x+y)−(L+M)∥≤∥x−L∥+∥y−M∥\|(x+y)-(L+M)\| \le \|x-L\| + \|y-M\|, which is post #9’s interchange law wearing a norm) and the ε/2 split. lim_scale has a classic trap: you want δ=ε/|a|\delta = \varepsilon / |a|, but aa might be zero, and “is aa zero?” is again not decidable. The repo’s route pads the denominator — δ=ε/(1+|a|)\delta = \varepsilon / (1 + |a|), positive unconditionally — and strictness comes out of the padding itself: |a|⋅t<(1+|a|)δ=ε|a| \cdot t < (1+|a|)\,\delta = \varepsilon whenever t<δt < \delta. No case split, no classical axiom. (lim_neg is then a one-liner: negation is scaling by −1-1.)
