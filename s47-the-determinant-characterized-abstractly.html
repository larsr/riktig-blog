The determinant, characterized abstractly, and multiplied for any field

A linear map from a space to itself can, once a basis is chosen, be written down as a grid of numbers, and that grid can be fed through a recursive computation that produces one scalar: the determinant. Choosing a different basis gives a different grid, but earlier work showed the resulting scalar comes out the same either way — an argument that rested on one further fact, itself simply assumed at the time: that the determinant of a composed map is the product of the two maps’ individual determinants.

That leaves two questions unresolved. First: can “how much a linear map scales volume, and whether it reverses orientation” be pinned down by a short list of properties that number has to have — the way “there is an identity element” and “the operation is associative” pin down what a group’s multiplication has to do, without reference to any one particular group — rather than by a step-by-step procedure for computing it? And if such a list of properties can be written down, does anything actually have them, or could the list turn out to describe nothing at all? Second: is the multiplication fact used before actually true — for every number system in which “linear map” and “scalar” make sense, not only the number systems well-behaved enough to test two numbers for equality, which is what the earlier argument secretly needed?

Pinning the number down by its own properties

Three properties answer the first question, each one stated without saying anything about how you would compute a value with them. A form that eats as many vectors as the space has dimensions is multilinear if scaling or adding in one argument position, with every other position held fixed, scales or adds the output the same way — “linear in each direction separately,” stated by replacing one argument at a time:

Record MultilinearForm (n : nat) (F V : Type) {FF : Field F}
    {VV : VectorSpace F V} := {
  mf_fun :> (Fin.t n -> V) -> F;
  #[global] mf_proper :: Proper (@famEq V _ n ==> equ) mf_fun;
  mf_add : forall (vs : Fin.t n -> V) (i : Fin.t n) (x y : V),
    mf_fun (upd vs i (x + y)) == mf_fun (upd vs i x) + mf_fun (upd vs i y);
  mf_scale : forall (vs : Fin.t n -> V) (i : Fin.t n) (a : F) (x : V),
    mf_fun (upd vs i (a *: x)) == a * mf_fun (upd vs i x)
}.

upd vs i x is the family vs with position i replaced by x; mf_add and mf_scale are exactly the additivity and homogeneity statements above, one position at a time. A form is alternating if feeding it two equal arguments in two different positions always produces 00 — “the measurement degenerates the moment two of the directions being compared coincide”:

Record Alternating {n : nat} {F V : Type} {FF : Field F}
    {VV : VectorSpace F V} (m : MultilinearForm n F V) := {
  alt_repeat : forall (vs : Fin.t n -> V) (i j : Fin.t n),
    i <> j -> vs i == vs j -> m vs == 0
}.

A determinant is one further requirement stacked on top of both: alternating, multilinear, and equal to exactly 11 on a fixed reference set of directions (a chosen basis BB) — the piece that turns “some scaling factor” into “the specific scaling factor a particular map applies,” the way fixing what “11 metre” means turns “a length” into a number you can compare across rulers.

Record Determinant (n : nat) (F V : Type) {FF : Field F}
    {VV : VectorSpace F V} (B : Basis n F V) := {
  det_mf :> MultilinearForm n F V;
  det_alt : Alternating det_mf;
  det_basis_1 : det_mf (basis_vec B) == 1
}.

det_mf and det_alt are the first two properties from above, packaged together; det_basis_1 is the third, basis_vec B being the family of vectors the chosen basis consists of. Nothing in this record mentions a matrix, an entry, a sign, or a recursive computation. It is data, not a search: a Determinant is a choice of such a form, supplied by whoever builds one, the same way a basis is a chosen list of vectors rather than something automatic instance resolution goes looking for on its own.

Does anything actually have these properties?

A list of properties with nothing satisfying it describes an empty set. The recursive-computation determinant from earlier work — call it detM, the one that expands along a matrix’s first column and runs on rational numbers by plain computation — needs to be checked against every field of the record above: linear in each row, alternating, and equal to 11 on the identity matrix (the standard basis, read as a matrix). Take multilinearity in one row first — additivity, precisely:

det⁡(A with row i replaced by x+y)=det⁡(A with row i=x)+det⁡(A with row i=y).\det(A \text{ with row } i \text{ replaced by } x+y) \;=\; \det(A \text{ with row } i = x) + \det(A \text{ with row } i = y).

  Lemma detM_mf_add : forall (n : nat) (vs : Fin.t n -> Fin.t n -> F)
      (i : Fin.t n) (x y : Fin.t n -> F),
    detM (upd vs i (x + y)) == detM (upd vs i x) + detM (upd vs i y).
  Proof.
    intros n; destruct n as [| p]; intros vs i x y.
    - exact (Fin.case0
               (fun i0 : Fin.t 0 =>
                  detM (upd vs i0 (x + y)) == detM (upd vs i0 x) + detM (upd vs i0 y))
               i).
    - rewrite (detM_upd_eq_upd_row (S p) vs i (x + y)),
        (detM_upd_eq_upd_row (S p) vs i x), (detM_upd_eq_upd_row (S p) vs i y).
      exact (det_rowI_add p i vs x y).
  Qed.

The dimension-00 case is vacuous (there is no i : Fin.t 0 to point at, so Fin.case0 disposes of it outright); in dimension n=Spn = S\,p, the one-slot family update upd used by the abstract record and the row-surgery operation upd_row used by the concrete matrix code are shown to agree pointwise (detM_upd_eq_upd_row), and once they are rewritten to match, an already-proved row-additivity lemma det_rowI_add closes the goal directly — a translation between two equivalent ways of describing “replace one row,” not new mathematics. Scaling is the same shape, one lemma over, and the alternating property comes straight from an already-proved lemma detM_alternating. Assembled, detM becomes a value of the Determinant record:

  (* THE INSTANCE: detM satisfies the abstract Determinant spec. *)
  Definition detM_Determinant (n : nat) :
      Determinant n F (Fin.t n -> F) (FinVec_Basis n) :=
    @Build_Determinant n F (Fin.t n -> F) FF _ (FinVec_Basis n)
      (detM_mf n) (detM_alt n) (detM_basis_1 n).

and Coq accepts a small resolution check confirming the pieces fit together unaided:

Example detM_determinant_check (F : Type) (FF : Field F) (n : nat) :
  Determinant n F (Fin.t n -> F) (FinVec_Basis n) := detM_Determinant n.

The list of properties is satisfiable, and satisfiable by exactly the computation anyone would expect it to be.

A theorem that costs nothing extra: swapping two arguments flips the sign

Because Determinant is a real record with real fields, anything provable from mf_add and alt_repeat alone is true of every determinant, on every field, for every basis — without ever touching detM. Here is one such fact: swapping the values an alternating multilinear form reads at two distinct positions negates the form’s value.

m(…,b⏟i,…,a⏟j,…)=−m(…,a⏟i,…,b⏟j,…),a=vi,b=vj.m(\dots, \underbrace{b}_{i}, \dots, \underbrace{a}_{j}, \dots) \;=\; -\,m(\dots, \underbrace{a}_{i}, \dots, \underbrace{b}_{j}, \dots), \qquad a = v_i,\ b = v_j.

The proof is one trick, run twice. Set WW to be vv with both positions ii and jj forced to a+ba+b; since WW then has two distinct positions holding the same value, alt_repeat gives m(W)=0m(W) = 0 outright. Expanding that 00 by additivity, first in position jj, then in position ii, and writing only the two active slots:

0=m(…,a+b,…,a+b,…){two equal slots: alternating}=m(…,a+b,…,a,…)+m(…,a+b,…,b,…){additivity in slot j}=m(…,a,…,a,…)+m(…,b,…,a,…)+m(…,a,…,b,…)+m(…,b,…,b,…){additivity in slot i}=m(…,b,…,a,…)+m(…,a,…,b,…){repeated-slot terms vanish: alternating}\begin{aligned} 0 &= m(\ldots, a+b, \ldots, a+b, \ldots) && \{\text{two equal slots: alternating}\}\\ &= m(\ldots, a+b, \ldots, a, \ldots) + m(\ldots, a+b, \ldots, b, \ldots) && \{\text{additivity in slot } j\}\\ &= m(\ldots, a, \ldots, a, \ldots) + m(\ldots, b, \ldots, a, \ldots)\\ &\quad + m(\ldots, a, \ldots, b, \ldots) + m(\ldots, b, \ldots, b, \ldots) && \{\text{additivity in slot } i\}\\ &= m(\ldots, b, \ldots, a, \ldots) + m(\ldots, a, \ldots, b, \ldots) && \{\text{repeated-slot terms vanish: alternating}\} \end{aligned}

The two surviving terms are mm of the swapped family and m(v)m(v) itself, and their sum being zero makes each the negative of the other.

  Theorem mf_swap_neg : forall (vs : Fin.t n -> V) (i j : Fin.t n),
    i <> j -> m (fswap vs i j) == - m vs.
  Proof.
    intros vs i j Hij.
    pose (a := vs i). pose (b := vs j).
    pose (VS' := upd vs i (a + b)).
    assert (Hji : j <> i) by (intro Heq; apply Hij; symmetry; exact Heq).
    (* m (upd VS' j a) + m (upd VS' j b) == 0, from alt_repeat at W. *)
    assert (HW0 : m (upd VS' j a) + m (upd VS' j b) == 0).
    { assert (HWi : upd VS' j (a + b) i == a + b)
        by (rewrite (u_off VS' j i (a + b) Hji); unfold VS'; exact (u_at vs i (a+b))).
      assert (HWj : upd VS' j (a + b) j == a + b) by exact (u_at VS' j (a+b)).
      assert (HW0' : m (upd VS' j (a + b)) == 0).
      { apply (alt_repeat Halt (upd VS' j (a + b)) i j Hij). rewrite HWi, HWj. reflexivity. }
      rewrite <- HW0'.
      symmetry.
      exact (mf_add m VS' j a b). }
    (* Second summand m (upd VS' j b) == m vs. *)
    assert (HP2val : m (upd VS' j b) == m vs).
    { assert (HVS'j : b == VS' j) by (unfold VS'; symmetry; exact (u_off vs i j (a+b) Hij)).
      assert (HP2 : forall k, upd VS' j b k == VS' k) by (intros k; exact (u_idem VS' j b HVS'j k)).
      assert (Hm_P2 : m (upd VS' j b) == m VS') by (apply (mf_proper m); exact HP2).
      assert (Hexp2 : m VS' == m (upd vs i a) + m (upd vs i b))
        by exact (mf_add m vs i a b).
      assert (HVSia : forall k, upd vs i a k == vs k)
        by (intros k; exact (u_idem vs i a (reflexivity a) k)).
      assert (Hm1 : m (upd vs i a) == m vs) by (apply (mf_proper m); exact HVSia).
      assert (Hm2 : m (upd vs i b) == 0).
      { apply (alt_repeat Halt (upd vs i b) i j Hij).
        rewrite (u_at vs i b), (u_off vs i j b Hij). reflexivity. }
      rewrite Hm_P2, Hexp2, Hm1, Hm2.
      exact (mon_idr (m vs)). }
    (* First summand m (upd VS' j a) == m (fswap vs i j). *)
    assert (HP1val : m (upd VS' j a) == m (fswap vs i j)).
    { assert (Hreorder : forall k, upd VS' j a k == upd (upd vs j a) i (a + b) k).
      { intros k. unfold VS'. exact (u_comm vs i j (a+b) a Hij k). }
      assert (Hm_P1 : m (upd VS' j a) == m (upd (upd vs j a) i (a + b)))
        by (apply (mf_proper m); exact Hreorder).
      assert (Hexp3 : m (upd (upd vs j a) i (a + b))
                    == m (upd (upd vs j a) i a) + m (upd (upd vs j a) i b))
        by exact (mf_add m (upd vs j a) i a b).
      assert (HT1 : m (upd (upd vs j a) i a) == 0).
      { apply (alt_repeat Halt (upd (upd vs j a) i a) i j Hij).
        rewrite (u_at (upd vs j a) i a), (u_off (upd vs j a) i j a Hij), (u_at vs j a).
        reflexivity. }
      assert (HT2 : forall k, upd (upd vs j a) i b k == fswap vs i j k).
      { intros k.
        destruct (dec (i = k)) as [Hik | Hik].
        - rewrite <- Hik, (u_at (upd vs j a) i b).
          symmetry. exact (fswap_at_i vs i j).
        - destruct (dec (j = k)) as [Hjk | Hjk].
          + rewrite <- Hjk, (u_off (upd vs j a) i j b Hij), (u_at vs j a).
            symmetry. exact (fswap_at_j vs i j Hij).
          + rewrite (u_off (upd vs j a) i k b Hik), (u_off vs j k a Hjk).
            symmetry. unfold fswap.
            rewrite (swap_fin_ne n i j k (fun Heq => Hik (eq_sym Heq)) (fun Heq => Hjk (eq_sym Heq))).
            reflexivity. }
      assert (HT2m : m (upd (upd vs j a) i b) == m (fswap vs i j))
        by (apply (mf_proper m); exact HT2).
      rewrite Hm_P1, Hexp3, HT1, HT2m.
      exact (mon_idl (m (fswap vs i j))). }
    (* Assemble: m (upd VS' j a) == - m (upd VS' j b) (from HW0), then
       substitute both known values. *)
    assert (Hneg : m (upd VS' j a) == - m (upd VS' j b))
      by exact (neg_unique_l (m (upd VS' j b)) (m (upd VS' j a)) HW0).
    rewrite HP1val, HP2val in Hneg.
    exact Hneg.
  Qed.

Line for line, this is the trick just described: HW0 records that m(first summand)+m(second summand)=0m(\text{first summand}) + m(\text{second summand}) = 0, read off alt_repeat at WW; HP2val shows the second summand reduces to m(v)m(v) — an inner update that turns out to be idempotent, feeding mf_add again, with one of its two pieces sent to zero by alt_repeat applied to a fresh equal pair; HP1val shows the first summand reduces, symmetrically, to mm of the swapped family; and the closing assert Hneg reads the two known values back into HW0’s equation to conclude m(swap)=−m(v)m(\text{swap}) = -m(v). Read as a fact about a determinant specifically, this is immediate — plug in its own multilinear form and its own alternating witness:

Theorem det_swap_neg : forall (n : nat) (F V : Type) {FF : Field F}
    {VV : VectorSpace F V} (B : Basis n F V) (D : Determinant n F V B)
    (vs : Fin.t n -> V) (i j : Fin.t n),
  i <> j -> D (fswap vs i j) == - D vs.
Proof.
  intros n F V FF VV B D vs i j Hij.
  exact (mf_swap_neg (det_mf D) (det_alt D) vs i j Hij).
Qed.

Nothing about detM, no first column, no minors: this follows from the three properties alone, for any structure matching them.

The determinant multiplies, for every field

Here is the theorem that settles the second question left open at the start: det⁡(AB)=det⁡(A)⋅det⁡(B)\det(AB) = \det(A) \cdot \det(B), for square matrices over any field — no assumption that the field’s equality can be decided, which was exactly what the earlier argument needed and never justified. The obvious route — write out the determinant as a sum over all n!n! permutations and match up terms from the product — needs a fact about permutation signs (that composing two permutations multiplies their signs) that takes real work to set up. There is a shorter route, the classical argument for uniqueness of an alternating multilinear form, and it goes through two much smaller pieces instead.

Piece one: any row-additive, alternating function is antisymmetric under swapping two rows. This does not need detM at all — it is a fact about any function gg from matrices to the field that respects row-additivity and vanishes on repeated rows. Write u,vu, v for rows i,ji, j of AA; feeding gg the matrix with rows ii and jj both set to u+vu+v gives 00 (repeated rows), and expanding that 00 by additivity, first in row jj then in row ii, leaves exactly g(A)+g(A with rows i,j swapped)g(A) + g(\text{$A$ with rows $i,j$ swapped}), with the other two terms sent to zero along the way by the same repeated-row rule — the same (u+v)(u+v)-in-both-slots chain displayed for mf_swap_neg above, now stated directly for row surgery on matrices rather than for the positions of an abstract form. In compressed form:

0=g(M1)=g(A)+g(A with rows i,j swapped)⇒g(swapped)=−g(A).0 \;=\; g(M_1) \;=\; g(A) + g(A \text{ with rows } i,j \text{ swapped}) \;\Longrightarrow\; g(\text{swapped}) = -\,g(A).

  Theorem g_swap_antisym : forall (n:nat) (g : Matrix n n F -> F),
    Proper (equ ==> equ) g ->
    (forall (i:Fin.t n)(A:Matrix n n F)(r1 r2:Fin.t n->F),
       g (upd_row A i (fun k=>r1 k+r2 k)) == g(upd_row A i r1)+g(upd_row A i r2)) ->
    (forall (A:Matrix n n F)(i j:Fin.t n), i<>j -> (forall c, A i c==A j c) -> g A == 0) ->
    forall (A:Matrix n n F)(i j:Fin.t n), i<>j -> g (row_swap A i j) == - g A.
  Proof.
    intros n g Hproper Hadd Halt A i j Hij.
    assert (Hji : j <> i) by (intros Heq; apply Hij; exact (eq_sym Heq)).
    set (u := A i). set (v := A j).
    set (N1 := upd_row A j (fun k=>u k+v k)).
    set (M1 := upd_row N1 i (fun k=>u k+v k)).
    (* rows i, j of M1 both equal u + v, so g M1 == 0 by alternating. *)
    assert (HM1i : forall c, M1 i c == u c + v c).
    { intros c. exact (upd_row_at n N1 i (fun k=>u k+v k) c). }
    assert (HM1j : forall c, M1 j c == u c + v c).
    { intros c.
      transitivity (N1 j c).
      - exact (upd_row_ne n N1 i j (fun k=>u k+v k) c Hji).
      - exact (upd_row_at n A j (fun k=>u k+v k) c). }
    assert (HM1zero : g M1 == 0).
    { apply (Halt M1 i j Hij). intros c. rewrite (HM1i c), (HM1j c). reflexivity. }
    (* M1 expands, in row i, into g (upd_row N1 i u) + g (upd_row N1 i v). *)
    assert (HM1exp : g M1 == g (upd_row N1 i u) + g (upd_row N1 i v)).
    { exact (Hadd i N1 u v). }
    (* g (upd_row N1 i u) == g A: N1's row i is already u, so this is
       g N1 itself, and N1 expands (row j) into g A plus a zero term
       (rows i, j both u). *)
    assert (Hleft : g (upd_row N1 i u) == g A).
    { assert (HN1i : forall c, N1 i c == u c).
      { intros c. exact (upd_row_ne n A j i (fun k=>u k+v k) c Hij). }
      assert (Heq1 : upd_row N1 i u == upd_row N1 i (N1 i)).
      { apply upd_row_change_pointwise. intros k. symmetry. exact (HN1i k). }
      assert (Heq2 : upd_row N1 i (N1 i) == N1).
      { exact (upd_row_self n N1 i). }
      rewrite Heq1, Heq2.
      unfold N1.
      transitivity (g (upd_row A j u) + g (upd_row A j v)).
      - exact (Hadd j A u v).
      - assert (H1 : g (upd_row A j u) == 0).
        { apply (Halt (upd_row A j u) i j Hij). intros c.
          transitivity (A i c).
          - exact (upd_row_ne n A j i u c Hij).
          - transitivity (u c); [reflexivity |].
            symmetry. exact (upd_row_at n A j u c). }
        assert (H2 : g (upd_row A j v) == g A).
        { assert (HAjv : upd_row A j v == A).
          { assert (Hveq : forall c, v c == A j c) by (intros c; reflexivity).
            transitivity (upd_row A j (A j)).
            - apply upd_row_change_pointwise. exact Hveq.
            - exact (upd_row_self n A j). }
          apply Hproper. exact HAjv. }
        rewrite H1, H2.
        exact (mon_idl (g A)). }
    (* g (upd_row N1 i v) == g (row_swap A i j): reorder the two updates
       (they commute, i <> j), then expand row j; one term IS the swap,
       the other has rows i, j both v (killed by alternating). *)
    assert (Hright : g (upd_row N1 i v) == g (row_swap A i j)).
    { assert (Hcomm : upd_row N1 i v == upd_row (upd_row A i v) j (fun k=>u k+v k)).
      { unfold N1. exact (upd_row_comm n A j i (fun k=>u k+v k) v Hji). }
      rewrite Hcomm.
      set (A2 := upd_row A i v).
      transitivity (g (upd_row A2 j u) + g (upd_row A2 j v)).
      - exact (Hadd j A2 u v).
      - assert (H1 : g (upd_row A2 j u) == g (row_swap A i j)).
        { assert (Heq3 : upd_row A2 j u == row_swap A i j).
          { unfold A2. exact (upd_row_row_swap_eq n A i j Hij). }
          apply Hproper. exact Heq3. }
        assert (H2 : g (upd_row A2 j v) == 0).
        { apply (Halt (upd_row A2 j v) i j Hij). intros c.
          transitivity (v c).
          - transitivity (A2 i c).
            + exact (upd_row_ne n A2 j i v c Hij).
            + exact (upd_row_at n A i v c).
          - symmetry. exact (upd_row_at n A2 j v c). }
        rewrite H1, H2.
        exact (mon_idr (g (row_swap A i j))). }
    assert (HM1full : g M1 == g A + g (row_swap A i j)).
    { rewrite HM1exp, Hleft, Hright. reflexivity. }
    assert (Hsum : g A + g (row_swap A i j) == 0).
    { transitivity (g M1); [symmetry; exact HM1full | exact HM1zero]. }
    exact (neg_unique_r (g A) (g (row_swap A i j)) Hsum).
  Qed.

M1 is the matrix with rows ii and jj both forced to u+vu+v, so g M1 == 0 by hypothesis (HM1zero); expanding it in row ii (HM1exp) gives two pieces, and each is chased down separately — Hleft shows the piece with row ii restored to uu collapses back to g A, Hright shows the piece with row ii set to vv is, after reordering the two independent row updates and expanding once more in row jj, exactly g of the swapped matrix. Adding the two known values and cancelling gives the antisymmetry. This is proved once, about an arbitrary such gg — a lemma about row-additive alternating functions in general, with no reference to detM.

Piece two: uniqueness. Any row-additive, homogeneous, alternating function gg on n×nn \times n matrices agrees with detM, up to the single scalar g(I)g(I):

g(A)=g(I)⋅det⁡(A).g(A) \;=\; g(I) \cdot \det(A).

The proof expands AA’s rows as linear combinations of standard basis rows (row_expand, reused for both gg and detM, since it needs only row-additivity and homogeneity), reducing the claim to matching gg and g(I)⋅detg(I)\cdot\det on every matrix whose rows are drawn from the standard basis. On such a matrix, indexed by a function f:Finn→Finnf : \mathrm{Fin}\ n \to \mathrm{Fin}\ n: if ff repeats a value, two rows coincide and both sides are 00 by the alternating property (basis_rows_g_eq_detM, the “repeat” branch); otherwise one swap of positions strictly reduces the count of positions where ff disagrees with the identity (piece one gives the antisymmetry the induction needs at each swap), and the induction bottoms out at f=idf = \mathrm{id}, where both sides are visibly g(I)⋅1g(I)\cdot 1. Injectivity of ff is never assumed or needed — the two cases (a repeat, or a swap that strictly shrinks the count) are exhaustive on their own.

  Theorem g_row_expand_eq : forall (N:nat) (g : Matrix N N F -> F),
    Proper (equ ==> equ) g ->
    (forall (i:Fin.t N)(A:Matrix N N F)(r1 r2:Fin.t N->F),
       g (upd_row A i (fun k=>r1 k+r2 k)) == g(upd_row A i r1)+g(upd_row A i r2)) ->
    (forall (i:Fin.t N)(A:Matrix N N F)(c:F)(r0:Fin.t N->F),
       g (upd_row A i (fun k=>c*r0 k)) == c*g(upd_row A i r0)) ->
    (forall (A:Matrix N N F)(i j:Fin.t N), i<>j -> (forall c, A i c==A j c) -> g A == 0) ->
    forall A : Matrix N N F, g A == g mat_id * detM A.
  Proof.
    intros N g Hproper Hadd Hscale Halt A.
    assert (Hg_exp := row_expand N N g Hproper Hadd Hscale A).
    assert (Hd_exp := det_row_expand N A).
    assert (Hpoint : forall f : Fin.t N -> Fin.t N,
      prodF (fun i => A i (f i)) * g (basis_rowsP f)
        == g mat_id * (prodF (fun i => A i (f i)) * detM (basis_rowsP f))).
    { intros f.
      transitivity (prodF (fun i => A i (f i)) * (g mat_id * detM (basis_rowsP f))).
      - apply ring_mul_proper; [reflexivity |].
        exact (basis_rows_g_eq_detM N g Hproper Hadd Halt f).
      - transitivity ((prodF (fun i => A i (f i)) * g mat_id) * detM (basis_rowsP f)).
        + symmetry. exact (ring_mul_assoc (prodF (fun i => A i (f i))) (g mat_id) (detM (basis_rowsP f))).
        + transitivity ((g mat_id * prodF (fun i => A i (f i))) * detM (basis_rowsP f)).

Hg_exp and Hd_exp are row_expand applied to gg and (via det_row_expand) to detM itself — both write their argument as a sum, over every function ff from row-indices to row-indices, of an entry-product times the value at the standard-basis matrix EfE_f that ff picks out. On the blackboard the whole theorem is the chain

g(A)=∑f(∏iAi,f(i))g(Ef){row expansion of g}=∑f(∏iAi,f(i))g(I)det⁡(Ef){the induction on standard-basis matrices}=g(I)∑f(∏iAi,f(i))det⁡(Ef){factor the constant out of the sum}=g(I)det⁡(A){row expansion of det⁡, read backwards}\begin{aligned} g(A) &= \sum_{f} \Bigl(\prod_i A_{i,f(i)}\Bigr)\, g(E_f) && \{\text{row expansion of } g\}\\ &= \sum_{f} \Bigl(\prod_i A_{i,f(i)}\Bigr)\, g(I)\,\det(E_f) && \{\text{the induction on standard-basis matrices}\}\\ &= g(I) \sum_{f} \Bigl(\prod_i A_{i,f(i)}\Bigr)\, \det(E_f) && \{\text{factor the constant out of the sum}\}\\ &= g(I)\,\det(A) && \{\text{row expansion of } \det, \text{ read backwards}\} \end{aligned}

Hpoint is the second step, where the induction (basis_rows_g_eq_detM) enters: term by term, gg and g(I)⋅detg(I)\cdot\det agree on every one of those standard-basis matrices. fsum_ext turns that termwise agreement into agreement of the sums, fsum_scale is the third step, factoring g(I)g(I) out, and Hd_exp read backwards is the last.

Assembling the product law. Fix BB and apply piece two to g(A):=det⁡(AB)g(A) := \det(AB) — which is row-additive, homogeneous, and alternating in AA, because row ii of A⋅BA \cdot B is built from row ii of AA alone, linearly, and equal rows of AA force equal rows of A⋅BA\cdot B:

  Definition g_mulB {N:nat} (B : Matrix N N F) (A : Matrix N N F) : F := detM (mat_mul A B).

g_mulB needs to be shown row-additive, homogeneous, and alternating before piece two applies to it. The first two follow because row ii of A⋅BA\cdot B is built from row ii of AA alone, linearly, through BB — an algebraic fact about matrix multiplication (mat_mul_upd_row, not shown here) feeding the already-proved row-additivity and row-scaling lemmas for detM. The third, that equal rows of AA force g_mulB to vanish, is a two-line consequence of detM’s own alternating property — equal rows in, equal rows out:

Ai=Aj⇒(AB)i=(AB)j⇒det⁡(AB)=0.A_i = A_j \;\Longrightarrow\; (AB)_i = (AB)_j \;\Longrightarrow\; \det(AB) = 0.

  Lemma g_mulB_alt : forall (N:nat)(B:Matrix N N F)(A:Matrix N N F)(i j:Fin.t N),
    i<>j -> (forall c, A i c == A j c) -> g_mulB B A == 0.
  Proof.
    intros N B A i j Hij Heq. unfold g_mulB.
    apply (detM_alternating N (mat_mul A B) i j Hij).
    intros c. unfold mat_mul. apply bigopF_ext. intros l.
    apply ring_mul_proper; [exact (Heq l) | reflexivity].
  Qed.

Uniqueness then finishes it in three steps:

det⁡(AB)=det⁡(I⋅B)⋅det⁡(A){piece two, at g(A):=det⁡(AB),g(I)=det⁡(I⋅B)}=det⁡(A)⋅det⁡(I⋅B){commutativity of the field multiplication}=det⁡(A)⋅det⁡(B){I⋅B=B}\begin{aligned} \det(AB) &= \det(I \cdot B)\cdot\det(A) && \{\text{piece two, at } g(A) := \det(AB),\ g(I) = \det(I \cdot B)\}\\ &= \det(A)\cdot\det(I \cdot B) && \{\text{commutativity of the field multiplication}\}\\ &= \det(A)\cdot\det(B) && \{I \cdot B = B\} \end{aligned}

  Theorem detM_mul : forall (N:nat)(A B:Matrix N N F), detM (mat_mul A B) == detM A * detM B.
  Proof.
    intros N A B.
    destruct N as [| p0].
    - transitivity (1:F).
      + exact (detM_0 (mat_mul A B)).
      + transitivity ((1:F)*1).
        * symmetry. exact (ring_mul_1l 1).
        * apply ring_mul_proper; symmetry; [exact (detM_0 A) | exact (detM_0 B)].
    - assert (Hg := g_row_expand_eq (S p0) (g_mulB B) (g_mulB_proper (S p0) B)
                 (fun i A' r1 r2 => g_mulB_add (S p0) p0 eq_refl B i A' r1 r2)
                 (fun i A' c r0 => g_mulB_scale (S p0) p0 eq_refl B i A' c r0)
                 (g_mulB_alt (S p0) B) A).
      unfold g_mulB in Hg.
      transitivity (detM (mat_mul mat_id B) * detM A).
      + exact Hg.
      + transitivity (detM A * detM (mat_mul mat_id B)).
        * exact (field_mul_comm (detM (mat_mul mat_id B)) (detM A)).
        * apply ring_mul_proper; [reflexivity |].
          apply detM_proper. exact (mat_mul_1l (S p0) (S p0) B).
  Qed.

Dimension 00 is a one-line special case (the determinant of any empty matrix is 11). Otherwise, Hg is g_row_expand_eq read at g_mulB B: it says det⁡(AB)=det⁡(I⋅B)⋅det⁡(A)\det(AB) = \det(I\cdot B)\cdot\det(A) directly. mat_mul_1l erases the leftover identity, and field_mul_comm puts the two factors in the expected order. No decidable equality anywhere — the whole route runs on Fin.t’s equality, which is always decidable regardless of what the field FF is, and on the field’s own axioms alone.

Running the arithmetic

Fix two 3×33\times 3 matrices over the rationals — a nonsingular one, mat3_ns_g, and a second, mat3_b_g. Both sides of the product law evaluate to the same number independently, by conversion, before the theorem above is ever invoked to relate them:

Example detM_mulGeneral_Q_compute_lhs :
  detM (mat_mul mat3_ns_g mat3_b_g) == ((-15)#1)%Q.
Proof. reflexivity. Qed.

Example detM_mulGeneral_Q_compute_rhs :
  detM mat3_ns_g * detM mat3_b_g == ((-15)#1)%Q.
Proof. reflexivity. Qed.

and applying detM_mul itself to this pair of matrices is a one-line call, standing in place of either computation:

Example detM_mulGeneral_Q :
  detM (mat_mul mat3_ns_g mat3_b_g) == detM mat3_ns_g * detM mat3_b_g.
Proof. exact (detM_mul 3 mat3_ns_g mat3_b_g). Qed.
