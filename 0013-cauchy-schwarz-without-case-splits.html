riktig #13 — Cauchy–Schwarz without case splits

Thirteenth post documenting the riktig project; the linear-algebra course thread continues alongside the opening of a new one: channels. Proofs shown blackboard-style and as they live in the repository.

What we are doing, and why. Verified software in Coq, built for modifiability. This wave adds the geometry an inner product buys you — Cauchy–Schwarz, projections — a determinant you can compute, and the first stone of an entirely different building: fault-tolerant channels for telecom protocols. Three unrelated-looking topics, one discipline: define the honest general statement, then prove it with only the tools the definition allows.

Positivity is a choice, not a given

Post #8 built InnerProduct without assuming ⟨u,u⟩≥0\langle u,u\rangle \ge 0 — deliberately, because Minkowski space-time’s inner product is indefinite, and we want that as a legal instance later. Today’s addition is the missing piece as an opt-in mixin (Interface/PosInner.v):

Class PosInner (F V : Type) ... := {
  pi_nonneg   : forall u : V, 0 <= ⟨ u , u ⟩;
  pi_definite : forall u : V, ⟨ u , u ⟩ == 0 -> u == 0
}.

Euclidean geometry is InnerProduct plus PosInner; Minkowski geometry is InnerProduct alone. One flag, two geometries, no duplicated machinery.

Cauchy–Schwarz — the theorem, and the trap

Blackboard, the fast way. For any u,vu, v and scalar tt: 0≤⟨u−tv,u−tv⟩=⟨u,u⟩−2t⟨u,v⟩+t2⟨v,v⟩.0 \le \langle u - tv, u - tv\rangle = \langle u,u\rangle - 2t\langle u,v\rangle + t^2\langle v,v\rangle. Minimize over tt by setting t=⟨u,v⟩/⟨v,v⟩t = \langle u,v\rangle / \langle v,v\rangle and simplify to get ⟨u,v⟩2≤⟨u,u⟩⟨v,v⟩\langle u,v\rangle^2 \le \langle u,u\rangle\langle v,v\rangle. ∎

Every textbook proof does exactly this — and every textbook proof divides by ⟨v,v⟩\langle v,v\rangle. If v=0v = 0, that division is nonsense, so the textbook adds a throat-clear: “if v≠0v \ne 0…” handled as a trivial separate case “…and if v=0v=0 both sides are zero.” Two cases, decided by asking “is ⟨v,v⟩\langle v,v\rangle zero?”

Here is the trap for us: that question is not decidable. Our scalars are an abstract ordered field — think computable reals — and “x=0x = 0 or x≠0x \ne 0” is exactly the law of excluded middle applied to something we cannot compute. A proof that case-splits on it isn’t a Coq proof of the general theorem; it’s a proof with an invisible extra hypothesis.

The constructive route: don’t ask, pad instead

The idea. Never test ⟨v,v⟩\langle v,v\rangle for zero. Instead prove a slightly weaker statement for every positive slack ε\varepsilon, then let ε→0\varepsilon \to 0 do the work that division was doing.

⟨u,v⟩2≤(⟨v,v⟩+ε)⟨u,u⟩for every ε>0.\langle u,v\rangle^2 \le \big(\langle v,v\rangle + \varepsilon\big)\langle u,u\rangle \qquad \text{for every } \varepsilon > 0.

This one is provable with a positive divisor sε=⟨v,v⟩+εs_\varepsilon = \langle v,v\rangle + \varepsilon, always safely positive (add_nonneg_pos, using only pi_nonneg — no test needed). Substitute t=⟨u,v⟩/sεt = \langle u,v\rangle / s_\varepsilon into the same blackboard expansion, multiply through by sεs_\varepsilon, and it falls out with no case split. In the repository, this padded version is its own named lemma (Theory/InnerGeometry.v):

Lemma cs_padded : forall (u v : V) (eps : F), 0 < eps ->
  ⟨u,v⟩ * ⟨u,v⟩ <= (norm2 v + eps) * norm2 u.

Then the general theorem is one squeeze away:

Theorem cauchy_schwarz : forall u v : V,
  ⟨ u , v ⟩ * ⟨ u , v ⟩ <= norm2 u * norm2 v.

using a new, reusable order lemma — “below b+cεb + c\varepsilon for every positive ε\varepsilon implies below bb” — proved once in Theory/Order.v:

le_of_le_add_all_eps_mul :
  0 <= c -> (forall eps, 0 < eps -> a <= b + eps * c) -> a <= b

This is the same squeeze technique post #11 used for limit uniqueness and post #12 used for the chain rule — Cauchy–Schwarz turns out to be an analysis theorem in algebra’s clothing, and the tree’s shared toolbox proves it for free.

The payoff: cauchy_schwarz in the repository carries no side condition. Not “v≠0v \ne 0” as a hypothesis, not a disjunction to case on afterward. Read the type and it is exactly the sentence from the blackboard — the constructive discipline cost real proof effort, and bought back the clean statement everyone actually wants to quote.

What it buys: projection

With Cauchy–Schwarz-grade machinery in hand, projecting uu onto a nonzero vv — projvu=⟨u,v⟩⟨v,v⟩v\mathrm{proj}_v u = \dfrac{\langle u,v\rangle}{\langle v,v\rangle}v — comes with its defining property for free:

Theorem proj_orthogonal : orthogonal (u + - proj v u) v.
Theorem best_approx : forall c, norm2 (u + - proj v u) <= norm2 (u + - (c *: v)).

The projection is the closest point on the line, and the residual is perpendicular to it — the Gram–Schmidt engine, one theorem short of orthogonal bases (queued: the loop over Fin n that iterates this per basis vector).

  • A new building begins (Channel/Trace.v): the telecom leg of riktig. A faulty channel is modeled as a relation between what was sent and what arrived — no probabilities, just possibility. lossy (dropped packets, as a subsequence relation), dup (duplication), lossy_dup (both — the wire a real retransmission protocol faces), reorder (via the standard library’s Permutation). The file proves these form a lattice — perfect ⊂ lossy ⊂ lossy+dup, perfect ⊂ reorder — and, just as importantly, proves the missing arrows really are missing: ~ lossy_dup [0;1] [1;0] (a wire that only drops and duplicates can never reorder).
