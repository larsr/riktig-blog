riktig #33 — the determinant is multiplicative

Thirty-third post in the series documenting the riktig project.

What we are doing, and why. We are building software whose correctness is proved in Coq. One thread of this series has chased a single classical theorem across seven attempts: det(A·B) = det A · det B for the computable Laplace-recursion determinant detM of post #14. Posts #23 through #28 built the classical ingredients one by one — the Leibniz reformulation, row-swap antisymmetry in full generality, the characteristic-free alternating fact, arbitrary-row linearity, and all three elementary-matrix identities — leaving exactly one gap: the composition argument that assembles them. Today it is assembled. Multiplicativity is proved, by certified Gaussian elimination, for every field with decidable equality — and unconditionally over ℚ, where it discharges the long-standing hypothesis of the coordinate-free determinant.

The theorem, and its one honest hypothesis

Theorem detM_mul_dec : forall (dec : DecEq F) (n : nat)
    (A B : Matrix n n F),
  detM (mat_mul A B) == detM A * detM B.

The DecEq F argument deserves a plain explanation, because it is a mathematical necessity, not a proof-engineering shortcut. The argument runs by Gaussian elimination, and elimination must at each step decide whether a pivot candidate is zero. In this library a Field carries no such decision procedure — field equality is a setoid ==, and for a general field there is no algorithm answering x == 0. So the theorem takes the decision procedure as an explicit argument, on exactly the two results that need it, and nothing else in the file assumes it. For any concrete field of interest the hypothesis is free: over ℚ,

Corollary detM_mul_Q : forall (n : nat) (A B : Matrix n n Q),
  detM (mat_mul A B) == detM A * detM B.

is unconditional.

The shape of the proof

Everything lives in Impl/DetMul.v. The engine is a certificate, not a computation on paper:

Theorem gauss_reduce : forall (dec : DecEq F) (n : nat) (A : Matrix n n F),
  exists l : list (ElemOp n),
    (eops_apply l A == mat_id)
    \/ (exists i : Fin.t n, forall j : Fin.t n, eops_apply l A i j == 0).

Every square matrix is taken, by a finite list of elementary row operations, either to the identity or to a matrix with an explicit zero row. No rank, no dimension theory, no case analysis left implicit — the disjunction is the singular/nonsingular dichotomy, carried as data. The elementary operations are proof-carrying values (ElemOp): a swap, a scale that carries a certified inverse (c*d == 1 is part of the value), or an add-multiple-of-a-row. Each kind changes the determinant by a known factor — packaged from the earlier posts’ det_elem_swap, det_rowI_scale, det_elem_add — and factors of a list multiply (det_eops_apply), the accumulated factor keeping a certified inverse of its own.

From the certificate, the headline theorem is two short stories:

  • A reduces to the identity. With f the accumulated factor, f · det A == 1; pushing the same operation list through A·B (they act on the left factor: eops_apply_mul) gives f · det(A·B) == det B. The product law follows by commutative regrouping of those two equations — notably, no division is ever performed.
  • A reduces to a zero row. A matrix with a zero row has zero determinant (det_zero_row — which is arbitrary-row scaling at c := 0, so it too is characteristic-free), and a zero row of A survives into A·B (mat_mul_zero_row), so both sides are 0 once the factor is cancelled through its certified inverse.

One standalone deliverable fell out on the way, the fact the roadmap had always predicted this argument would need:

Theorem det_upper_triangular : forall (n : nat) (A : Matrix n n F),
  (forall i j : Fin.t n, (fin_to_nat j < fin_to_nat i)%nat -> A i j == 0) ->
  detM A == prodF (fun i => A i i).

— the determinant of an upper-triangular matrix is the product of its diagonal, by induction: an upper-triangular first column has one possibly-nonzero entry, so the Laplace recursion collapses to a single term per level. (The final proof route through mat_id ended up not needing it; it is proved because it was promised, and it is true, unconditionally.)

The debt it pays off

Post #17 introduced — and posts since have honestly repeated — the one conditional corner of the linear-algebra tower: Theory/Determinant.v defines the determinant of an endomorphism coordinate-freely, det B f := detM (mat_of B B f), but its two headline consequences stood under an undischarged Context hypothesis: multiplicativity of detM. Over ℚ, that hypothesis is now discharged, and the conditional theorems become theorems:

Corollary det_comp_Q : forall (U : Type) (VU : VectorSpace Q U) (n : nat)
    (B : Basis n Q U) (f g : LinearMap Q U U),
  det B (lm_comp f g) == det B f * det B g.

Corollary det_basis_indep_Q : forall (U : Type) (VU : VectorSpace Q U)
    (n : nat) (B B' : Basis n Q U) (f : LinearMap Q U U),
  det B' f == det B f.

The determinant of a linear map on any ℚ-vector space, read through any basis, is multiplicative under composition and does not depend on the basis. That is the coordinate-free statement this whole thread was for.

Checked on actual numbers

As always, the theorems are also run. With the house-standard nonsingular matrix [[1,2,3],[4,5,6],[7,8,10]] (determinant −3) and a second matrix with determinant 5, both sides of the product law are evaluated independently by reflexivity — detM (A·B) computes to −15, detM A * detM B computes to −15 — before the theorem is applied to the same instance as a resolution check. The triangular fact gets its own evaluation: diagonal 2, 3, 4, determinant 24.

What was proved, plainly — and what remains

  • detM (A·B) == detM A · detM B for every field with decidable equality (detM_mul_dec), unconditionally over ℚ (detM_mul_Q), with det_comp_Q and det_basis_indep_Q as the coordinate-free payoff. Zero Admitted, zero Axiom, characteristic-free throughout.
  • Unconditional supporting theorems for any field: det_zero_row, mat_mul_zero_row, det_upper_triangular, and the proof-carrying elementary-operation layer with its factor calculus.
  • What remains, honestly: detM_mul without DecEq — the candidate route is a Leibniz/Cauchy–Binet-style expansion of detM (A·B), where the only case analysis is injectivity of maps on Fin n, decidable for free. Theory/Determinant.v’s Context hypothesis deliberately stays as the interface until that strengthening exists; it is now known to be dischargeable for every concrete field this repository uses.
