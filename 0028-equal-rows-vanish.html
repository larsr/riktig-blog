riktig #28 — equal rows vanish, without counting to two

Twenty-eighth post documenting the riktig project. Three results in one file — the last three ingredients detM_mul still needed — with the alternating property at the center, proved by a shorter route than the one planned.

What we are doing, and why. Verified software in Coq, built for modifiability. Post #27 proved general row-swap antisymmetry: swapping any two rows negates the determinant. What detM_mul — general multiplicativity, the fact this whole thread is building toward — still needed after that was three classical ingredients: the alternating property (“equal rows make the determinant vanish”), linearity in an arbitrary row, and the determinant of the third elementary matrix. This post gets all three. The first is the centerpiece: it looks like a free corollary of antisymmetry — and in most textbooks, it is. Here it wasn’t quite, and the reason is a genuine feature of how this library is built.

Why the obvious corollary doesn’t work

If swapping two equal rows negates the determinant, but swapping equal rows changes nothing, then det⁡A=−det⁡A\det A = -\det A, so 2det⁡A=02 \det A = 0. The textbook finishes with “and therefore det⁡A=0\det A = 0” — silently leaning on the field having characteristic other than 2, where 2=02 = 0 would make that step false. This codebase’s Field interface makes no such assumption, deliberately, since the very first determinant file: a field of characteristic 2 is a completely legal instance, and a proof that secretly required 2≠02 \ne 0 would be a proof that only worked for some fields while claiming to work for all of them.

The plan going into this post was accordingly more elaborate: pair up every permutation in the Leibniz sum with itself-postcomposed-by-a- fixed-transposition, and show the pairs cancel termwise — a genuine piece of new combinatorics, sketched but not yet attempted.

What actually worked

It turned out unnecessary. Rather than reasoning about permutations at all, the proof goes back to detM’s original definition — the cofactor recursion, not the Leibniz sum — and argues directly by induction on matrix size.

For two adjacent equal rows, the cofactor expansion has exactly two terms that involve those rows, with opposite signs (that much is just the recursion’s own alternating-sign structure, no theorem needed). The content is showing those two terms are not just opposite in sign but have the exact same magnitude: deleting either of two equal adjacent rows leaves an identical minor — the same matrix, entry for entry, no relabeling needed. Two terms, same value, opposite sign, cancel by plain group inverse. No factor of 2 anywhere, so nothing about characteristic ever enters the argument:

Theorem detM_alternating : forall (N : nat) (A : Matrix N N F) (i j : Fin.t N),
  i <> j -> (forall c, A i c == A j c) -> detM A == 0.

The general (non-adjacent) case doesn’t need a new induction either — it reduces to the adjacent case with a single ordinary row swap, reusing post #27’s detM_swap_general directly. Simpler than the sign-flip proof’s own route, which needed real induction on the distance between positions; here, one swap suffices because the value being proved is 0, and 0=−00 = -0 makes the sign of the swap irrelevant to the final answer.

The second ingredient: linearity in any row

Impl/Elim.v’s very first header — the file that started this whole five-attempt thread — named one thing it couldn’t reach: linearity of detM in an arbitrary row, not just the first. That fact had been sitting open since the beginning, and detM_mul needs it. It falls out now almost for free: to reason about row ii, conjugate it into the first position with a single swap — each side of the equation picks up the same negation sign, which cancels — then apply the row-1-specific facts that have existed since the start:

Theorem det_rowI_add : forall (p : nat) (i : Fin.t (S p)) (A : Matrix (S p) (S p) F)
    (r1 r2 : Fin.t (S p) -> F),
  detM (upd_row A i (fun k => r1 k + r2 k))
    == detM (upd_row A i r1) + detM (upd_row A i r2).

Theorem det_rowI_scale : forall (p : nat) (i : Fin.t (S p)) (A : Matrix (S p) (S p) F)
    (c : F) (r0 : Fin.t (S p) -> F),
  detM (upd_row A i (fun k => c * r0 k)) == c * detM (upd_row A i r0).

(upd_row A i r overwrites row ii of AA with the row function rr.) An old gap closed almost as a side effect of the swap machinery.

The third ingredient: adding a multiple of one row to another

Chaining the new row-ii linearity with the alternating fact gives the third elementary-matrix determinant identity in full:

Theorem det_elem_add : forall (n : nat) (i j : Fin.t n) (c : F) (A : Matrix n n F),
  i <> j -> detM (mat_mul (elem_add i j c) A) == detM A.

Adding a multiple of one row to another leaves the determinant unchanged — the third of the three classical row operations, alongside swap (post #27) and scale (much earlier in this thread).

Where this leaves detM_mul

Every classical building block general multiplicativity needs now exists: swap-antisymmetry, the alternating property, row multilinearity for any row, and all three elementary matrices’ determinant behavior. What’s left is purely the composition argument — the classical fact that any matrix reduces to a product of elementary matrices times an upper-triangular one, and that the determinant of an upper-triangular matrix is the product of its diagonal. Real work, not yet attempted, but for the first time in this thread, every ingredient is already on the shelf.
